01

설치 전 준비물

Claude Code를 설치하기 전에 딱 한 가지만 확인하면 됩니다 — Node.js v18 이상이 설치되어 있는지.

Node.js 버전 확인하기

Terminal
node --version # v18.0.0 이상이면 OK! # 예: v20.11.0, v22.1.0

설치 전 체크리스트

  • Node.js 18+ — 터미널에서 node --version으로 확인
  • 인터넷 연결 — 설치 파일 다운로드 및 인증에 필요
  • 터미널 접근 — macOS Terminal, Windows PowerShell, 또는 VS Code 터미널
💡
Node.js가 없다면? 이전 챕터 "터미널 기초"를 참고하세요. macOS는 brew install node, Windows는 nodejs.org에서 다운로드할 수 있습니다.
02

OS별 설치 방법

운영체제에 맞는 명령어 한 줄이면 설치 끝입니다.

🍎

macOS

Homebrew를 사용하여 설치합니다. 가장 간편하고 업데이트 관리도 쉽습니다.

🪟

Windows

Windows 패키지 매니저(winget)를 사용합니다. Windows 10 이상에서 기본 제공됩니다.

🐧

Linux

npm을 통해 글로벌 설치합니다. 모든 Linux 배포판에서 동일하게 사용 가능합니다.

🍎 macOS

Terminal
# Homebrew로 설치 brew install claude-code # 설치 확인 claude --version

🪟 Windows

PowerShell
# winget으로 설치 winget install Anthropic.ClaudeCode # 설치 확인 claude --version

🐧 Linux

Terminal
# npm으로 글로벌 설치 npm install -g @anthropic-ai/claude-code # 설치 확인 claude --version
💡
어떤 방법이든 결과는 같습니다. 세 가지 방법 모두 동일한 Claude Code CLI를 설치합니다. 자신의 OS에 맞는 방법을 사용하세요. npm 방법은 모든 OS에서 사용 가능한 범용 방법입니다.
03

로그인 & 인증

Claude Code를 사용하려면 인증이 필요합니다. 3가지 방법 중 하나를 선택하세요.

⭐

Claude.ai 구독 (추천)

Claude Pro / Team / Enterprise 구독이 있으면 가장 간단합니다. 별도의 API 키가 필요 없습니다.

🔑

Anthropic Console API

개발자용 API 키를 직접 사용합니다. 사용량 기반 과금으로 유연하게 쓸 수 있습니다.

☁️

AWS Bedrock / GCP Vertex

기업 환경에서 클라우드 제공업체를 통해 접근합니다. 기존 클라우드 인프라에 통합 가능합니다.

방법 1: Claude.ai 구독으로 로그인 (가장 쉬움)

1

claude 실행

터미널에서 claude를 입력하면 로그인 화면이 나옵니다.

Terminal
claude
2

브라우저에서 인증

자동으로 브라우저가 열리고, Claude.ai 계정으로 로그인하면 인증이 완료됩니다.

3

완료!

터미널로 돌아오면 바로 사용할 수 있습니다. 인증 정보는 로컬에 안전하게 저장됩니다.

방법 2: API 키 사용

Terminal
# 환경변수로 API 키 설정 export ANTHROPIC_API_KEY="sk-ant-..." # 이후 claude 실행하면 자동으로 API 키 인식 claude

방법 3: AWS Bedrock / GCP Vertex

Terminal
# AWS Bedrock 사용 시 claude --provider bedrock # GCP Vertex AI 사용 시 claude --provider vertex
🎯
어떤 걸 선택할지 모르겠다면? Claude Pro 구독($20/월)이 가장 간편합니다. API 키는 사용량에 따라 비용이 달라지고, 클라우드 제공업체는 기업 환경에 적합합니다.
04

첫 대화 해보기

설치와 인증이 끝났으니, 이제 Claude Code와 첫 번째 대화를 해봅시다.

1

프로젝트 폴더로 이동

Claude Code는 현재 폴더를 기준으로 동작합니다. 먼저 작업할 프로젝트 폴더로 이동하세요.

Terminal
# 프로젝트 폴더로 이동 (없으면 새로 만들기) mkdir my-project cd my-project
2

claude 실행

터미널에 claude만 입력하면 대화 모드가 시작됩니다.

Terminal
claude # Claude Code가 시작되면 이런 화면이 나옵니다: ╭────────────────────────────────────╮ │ Claude Code v1.x.x │ │ Type your message... │ ╰────────────────────────────────────╯
3

첫 번째 질문하기

아무거나 물어보세요! 예를 들어 간단한 HTML 페이지를 만들어달라고 해봅시다.

Claude Code
이 폴더에 간단한 HTML 페이지를 만들어줘 # Claude가 파일을 생성하겠다고 제안합니다 Claude: index.html 파일을 생성하겠습니다. [Allow] [Deny]
4

결과 확인

권한을 승인하면 Claude가 파일을 생성합니다. 폴더를 확인해보세요!

Terminal
# 생성된 파일 확인 ls index.html # 브라우저에서 열어보기 (macOS) open index.html
🚨
권한 승인이 나오면 Y를 누르세요. Claude Code는 파일을 생성하거나 수정하기 전에 항상 허락을 구합니다. 안전하다고 판단되면 Y(Allow)를 눌러 승인하세요. 원하지 않는 작업이면 N(Deny)을 누르면 됩니다.
05

IDE 연동

Claude Code는 CLI 자체만으로 강력하지만, 좋아하는 에디터와 함께 쓰면 더 편합니다.

💻

VS Code

공식 확장 프로그램으로 에디터 안에서 바로 Claude Code를 사용할 수 있습니다. 가장 인기 있는 조합입니다.

🔧

JetBrains

IntelliJ, WebStorm, PyCharm 등 JetBrains IDE에서 플러그인으로 연동할 수 있습니다.

📝

Vim / Neovim

터미널 기반 에디터 사용자라면 별도 설치 없이 터미널 분할로 바로 사용 가능합니다.

VS Code 확장 프로그램 설치

Terminal
# VS Code 마켓플레이스에서 설치 code --install-extension anthropic.claude-code # 또는 VS Code에서 Extensions 탭 열고 # "Claude Code" 검색 후 Install 클릭

JetBrains 플러그인 설치

JetBrains IDE
# Settings → Plugins → Marketplace # "Claude Code" 검색 후 Install 클릭 # 또는 터미널에서 IDE의 내장 터미널 열고 claude # CLI를 직접 사용해도 됩니다
💡
CLI 자체만으로도 충분합니다. IDE 연동은 선택사항입니다. 많은 개발자들이 터미널에서 claude만으로 모든 작업을 합니다. IDE 연동은 편의성을 높여줄 뿐, 필수가 아닙니다.
06

트러블슈팅 & FAQ

설치 중 문제가 생겼나요? 가장 흔한 문제와 해결 방법을 모았습니다.

이 에러는 Claude Code가 시스템 PATH에 등록되지 않았을 때 발생합니다.

해결 방법:
1. 터미널을 완전히 닫고 다시 열어보세요.
2. npm list -g @anthropic-ai/claude-code로 설치 여부를 확인하세요.
3. 설치가 안 되어 있다면 npm install -g @anthropic-ai/claude-code로 재설치하세요.
4. macOS에서 Homebrew로 설치했다면 brew link claude-code를 시도해보세요.
5. 여전히 안 된다면 셸 설정 파일(~/.zshrc 또는 ~/.bashrc)에 npm 글로벌 경로가 포함되어 있는지 확인하세요.
로그인 문제는 대부분 브라우저 또는 네트워크 설정과 관련이 있습니다.

확인 사항:
1. 기본 브라우저가 정상적으로 열리는지 확인하세요.
2. Claude.ai에 유효한 구독(Pro/Team/Enterprise)이 있는지 확인하세요.
3. VPN이나 프록시를 사용 중이라면 일시적으로 끄고 시도해보세요.
4. claude logout 후 claude를 다시 실행하여 재인증을 시도하세요.
5. API 키 방식이라면 키가 올바르게 설정되었는지 echo $ANTHROPIC_API_KEY로 확인하세요.
Claude Code를 사용하려면 다음 중 하나가 필요합니다:

· Claude Pro ($20/월) — 개인 사용자에게 추천. 충분한 사용량이 제공됩니다.
· Claude Team ($25/월/인) — 팀 협업이 필요할 때.
· Claude Enterprise — 대규모 조직용. 관리 기능 포함.
· API 키 — 사용량 기반 과금. 구독 없이도 사용 가능하지만 비용이 달라질 수 있습니다.

무료 플랜으로는 Claude Code를 사용할 수 없습니다.
네, 가능합니다. 다만 몇 가지 설정이 필요할 수 있습니다.

· Claude Code는 api.anthropic.com에 HTTPS(443 포트)로 통신합니다. 이 도메인을 허용 목록에 추가하세요.
· 프록시 환경이라면 HTTPS_PROXY 환경변수를 설정하세요.
· AWS Bedrock이나 GCP Vertex를 사용하면 회사 내부 네트워크만으로도 사용 가능합니다.
· 인증 시 브라우저 접근이 필요하므로, 최초 인증은 외부 네트워크에서 하고 이후에는 토큰이 로컬에 저장됩니다.
설치 방법에 따라 제거 방법이 다릅니다:

macOS (Homebrew):
brew uninstall claude-code

Windows (winget):
winget uninstall Anthropic.ClaudeCode

npm (모든 OS):
npm uninstall -g @anthropic-ai/claude-code

인증 정보도 삭제하려면 claude logout을 먼저 실행한 후 제거하세요.
