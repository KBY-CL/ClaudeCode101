05

IDE & м•ұ м—°лҸҷ м •лҰ¬

к°Ғ нҷҳкІҪмқҖ лҸҷмқјн•ң Claude Code м—”м§„м—җ м—°кІ°лҗҳлҜҖлЎң CLAUDE.md нҢҢмқј, м„Өм •, MCP м„ңлІ„к°Җ лӘЁл“  нҷҳкІҪм—җм„ң мһ‘лҸҷн•©лӢҲлӢӨ.

нҷҳкІҪ мЈјмҡ” нҠ№м§• м„Өм№ҳ л°©лІ•
Terminal CLI нҢҢмқј нҺём§‘, лӘ…л № мӢӨн–ү, git нҶөн•©, MCP, мһҗлҸҷнҷ” curl -fsSL https://claude.ai/install.sh | bash

Cursor VS Code нҷ•мһҘ н”„лЎңк·ёлһЁкіј лҸҷмқјн•ң кё°лҠҘ Cursorмҡ© м„Өм№ҳ л§ҒнҒ¬

Web м„Өм№ҳ л¶Ҳн•„мҡ”, мһҘмӢңк°„ мһ‘м—…, iOS м•ұ м—°лҸҷ claude.ai/code л°©л¬ё нӣ„ л°”лЎң мӮ¬мҡ©
💡
CLI мһҗмІҙл§ҢмңјлЎңлҸ„ м¶©л¶„н•©лӢҲлӢӨ. IDE м—°лҸҷмқҖ м„ нғқмӮ¬н•ӯмһ…лӢҲлӢӨ. л§ҺмқҖ к°ңл°ңмһҗл“Өмқҙ н„°лҜёл„җм—җм„ң claudeл§ҢмңјлЎң лӘЁл“  мһ‘м—…мқ„ н•©лӢҲлӢӨ. IDE м—°лҸҷкіј лҚ°мҠӨнҒ¬нҶұ м•ұмқҖ нҺёмқҳм„ұмқ„ лҶ’м—¬мӨ„ лҝҗ, н•„мҲҳк°Җ м•„лӢҷлӢҲлӢӨ.
06

нҠёлҹ¬лё”мҠҲнҢ… & FAQ

м„Өм№ҳ мӨ‘ л¬ём ңк°Җ мғқкІјлӮҳмҡ”? к°ҖмһҘ нқ”н•ң л¬ём ңмҷҖ н•ҙкІ° л°©лІ•мқ„ лӘЁм•ҳмҠөлӢҲлӢӨ.

мқҙ м—җлҹ¬лҠ” Claude Codeк°Җ мӢңмҠӨн…ң PATHм—җ л“ұлЎқлҗҳм§Җ м•Ҡм•ҳмқ„ л•Ң л°ңмғқн•©лӢҲлӢӨ.

н•ҙкІ° л°©лІ•:
1. н„°лҜёл„җмқ„ мҷ„м „нһҲ лӢ«кі  лӢӨмӢң м—ҙм–ҙліҙм„ёмҡ”.
2. claude doctorлҘј мӢӨн–үн•ҳм—¬ м„Өм№ҳ нғҖмһ…кіј лІ„м „мқ„ нҷ•мқён•ҳм„ёмҡ”.
3. л„ӨмқҙнӢ°лёҢ м„Өм№ҳмқҳ кІҪмҡ° ~/.local/binмқҙ PATHм—җ нҸ¬н•Ёлҗҳм–ҙ мһҲлҠ”м§Җ нҷ•мқён•ҳм„ёмҡ”.
4. HomebrewлЎң м„Өм№ҳн–ҲлӢӨл©ҙ brew link --cask claude-codeлҘј мӢңлҸ„н•ҙліҙм„ёмҡ”.
5. Windowsм—җм„ң Git Bash кІҪлЎңлҘј м§Җм •н•ҳл Өл©ҙ: $env:CLAUDE_CODE_GIT_BASH_PATH="C:\Program Files\Git\bin\bash.exe"
лЎңк·ёмқё л¬ём ңлҠ” лҢҖл¶Җл¶„ лёҢлқјмҡ°м Җ лҳҗлҠ” л„ӨнҠёмӣҢнҒ¬ м„Өм •кіј кҙҖл Ёмқҙ мһҲмҠөлӢҲлӢӨ.

нҷ•мқё мӮ¬н•ӯ:
1. кё°ліё лёҢлқјмҡ°м Җк°Җ м •мғҒм ҒмңјлЎң м—ҙлҰ¬лҠ”м§Җ нҷ•мқён•ҳм„ёмҡ”.
2. Claude.aiм—җ мң нҡЁн•ң кө¬лҸ…(Pro / Max / Team / Enterprise) лҳҗлҠ” Anthropic Console кі„м •мқҙ мһҲлҠ”м§Җ нҷ•мқён•ҳм„ёмҡ”.
3. VPNмқҙлӮҳ н”„лЎқмӢңлҘј мӮ¬мҡ© мӨ‘мқҙлқјл©ҙ мқјмӢңм ҒмңјлЎң лҒ„кі  мӢңлҸ„н•ҙліҙм„ёмҡ”.
4. claude logout нӣ„ claudeлҘј лӢӨмӢң мӢӨн–үн•ҳм—¬ мһ¬мқёмҰқмқ„ мӢңлҸ„н•ҳм„ёмҡ”.
Claude CodeлҘј мӮ¬мҡ©н•ҳл Өл©ҙ лӢӨмқҢ мӨ‘ н•ҳлӮҳк°Җ н•„мҡ”н•©лӢҲлӢӨ:

· Claude Pro — к°ңмқё мӮ¬мҡ©мһҗм—җкІҢ м¶”мІң. м¶©л¶„н•ң мӮ¬мҡ©лҹүмқҙ м ңкіөлҗ©лӢҲлӢӨ.
· Claude Max — лҚ” лҶ’мқҖ мӮ¬мҡ©лҹүмқҙ н•„мҡ”н•ң к°ңмқё мӮ¬мҡ©мһҗмҡ©.
· Claude Team — нҢҖ нҳ‘м—…мқҙ н•„мҡ”н•  л•Ң.
· Claude Enterprise — лҢҖк·ңлӘЁ мЎ°м§Ғмҡ©. кҙҖлҰ¬ кё°лҠҘ нҸ¬н•Ё.
· Anthropic Console — мӮ¬мҡ©лҹү кё°л°ҳ API кіјкёҲ. кө¬лҸ… м—ҶмқҙлҸ„ мӮ¬мҡ© к°ҖлҠҘ.

л¬ҙлЈҢ н”ҢлһңмңјлЎңлҠ” Claude CodeлҘј мӮ¬мҡ©н•  мҲҳ м—ҶмҠөлӢҲлӢӨ. мһҗм„ён•ң к°ҖкІ©мқҖ claude.com/pricingмқ„ нҷ•мқён•ҳм„ёмҡ”.
л„ӨмқҙнӢ°лёҢ м„Өм№ҳлҠ” кё°ліём ҒмңјлЎң мһҗлҸҷ м—…лҚ°мқҙнҠёлҗ©лӢҲлӢӨ. лӢӨмқҢ л°©лІ•мңјлЎң л№„нҷңм„ұнҷ”н•ҳкұ°лӮҳ мұ„л„җмқ„ ліҖкІҪн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

мһҗлҸҷ м—…лҚ°мқҙнҠё л№„нҷңм„ұнҷ”:
export DISABLE_AUTOUPDATER=1

лҰҙлҰ¬мҠӨ мұ„л„җ ліҖкІҪ (stable мұ„л„җ к¶ҢмһҘ - м•Ҫ 1мЈј м§Җм—°):
settings.jsonм—җ м¶”к°Җ: { "autoUpdatesChannel": "stable" }

лҳҗлҠ” /config → мһҗлҸҷ м—…лҚ°мқҙнҠё мұ„л„җм—җм„ң ліҖкІҪн•©лӢҲлӢӨ.

мҲҳлҸҷ м—…лҚ°мқҙнҠё:
claude update
л„Ө, к°ҖлҠҘн•©лӢҲлӢӨ. лӢӨл§Ң лӘҮ к°Җм§Җ м„Өм •мқҙ н•„мҡ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

· Claude CodeлҠ” api.anthropic.comм—җ HTTPS(443 нҸ¬нҠё)лЎң нҶөмӢ н•©лӢҲлӢӨ. мқҙ лҸ„л©”мқёмқ„ н—Ҳмҡ© лӘ©лЎқм—җ м¶”к°Җн•ҳм„ёмҡ”.
· н”„лЎқмӢң нҷҳкІҪмқҙлқјл©ҙ HTTPS_PROXY нҷҳкІҪліҖмҲҳлҘј м„Өм •н•ҳм„ёмҡ”.
· AWS BedrockмқҙлӮҳ GCP VertexлҘј мӮ¬мҡ©н•ҳл©ҙ нҡҢмӮ¬ лӮҙл¶Җ л„ӨнҠёмӣҢнҒ¬л§ҢмңјлЎңлҸ„ мӮ¬мҡ© к°ҖлҠҘн•©лӢҲлӢӨ.
· мқёмҰқ мӢң лёҢлқјмҡ°м Җ м ‘к·јмқҙ н•„мҡ”н•ҳлҜҖлЎң, мөңмҙҲ мқёмҰқмқҖ мҷёл¶Җ л„ӨнҠёмӣҢнҒ¬м—җм„ң н•ҳкі  мқҙнӣ„м—җлҠ” нҶ нҒ°мқҙ лЎңм»¬м—җ м ҖмһҘлҗ©лӢҲлӢӨ.
м„Өм№ҳ л°©лІ•м—җ л”°лқј м ңкұ° л°©лІ•мқҙ лӢӨлҰ…лӢҲлӢӨ:

л„ӨмқҙнӢ°лёҢ м„Өм№ҳ (macOS / Linux / WSL):
rm -f ~/.local/bin/claude && rm -rf ~/.local/share/claude
